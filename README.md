## 🎮 ブラウザで遊べる！物流最適化パズルゲーム

『とりあえず物流、最適化していいですか？』

「工場（🏭）から店舗（🏪）へ、トラックのルートを引いてください。ただし、ルートの交差は禁止。すべての空き地を埋めて、積載効率100%を目指してください」

ルールはシンプル。「同じ色の拠点同士を線で結ぶ」だけ。
しかし、ただ繋ぐだけではクリアできません。盤面のすべてのマスを埋め尽くす（Coverage 100%） ことが、この業務の絶対条件です。

- 直感操作: スマホでもPCでも、指一本・マウス一つでサクサク遊べます。
- 無限の業務: 毎回異なるマップを生成。Level 1（研修）から始まり、District（管区）が進むごとに難易度が上昇。無限に遊べます。
- AI最適化機能: どうしても解けない（納期に間に合わない）時は、「AI Optimization（👁）」ボタンで模範解答を表示可能。
